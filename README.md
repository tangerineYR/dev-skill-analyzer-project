# 신입 개발자 직무별 요구 기술 분석 프로젝트

본 프로젝트는 텍스트 형태의 채용 공고 데이터를 시각화하여, 막연한 추측이 아닌 데이터에 기반한 신입 개발자 진로 설계 기준을 제시하는 것을 목표로 합니다.

## 1. 문제 인식

데이터사이언스학을 전공하며 데이터를 다루는 법을 배우지만, 정작 학생들의 진로 선택은 데이터가 아닌 막연한 소문에 의존하고 있습니다. 수백 개의 채용 공고는 그저 거대한 텍스트의 나열일 뿐, 그 안에서 직무별 핵심 기술의 차이를 한눈에 파악하기 어렵기 때문입니다.

본 프로젝트는 ‘데이터 시각화’ 를 통해 정보의 비대칭을 해소하고, 예비 개발자들이 자신의 강점에 맞는 직무를 전략적으로 선택할 수 있도록 돕습니다.

## 2. 분석 데이터

- 데이터 소스: 국내 주요 IT 채용 플랫폼 ‘원티드(Wanted)’

- 수집 대상: '신입(경력 1년 미만)' 필터를 적용한 개발 직군 3개 직무(서버 개발자, 프론트엔드 개발자, 데이터 사이언티스트)

- 유효 데이터 수: 서버 개발자 44건, 프론트엔드 개발자 46건, 데이터 사이언티스트 29건 (총 119건)

## 3. 파일 설명

### scraper.py

Python의 Selenium과 BeautifulSoup을 활용한 웹 스크레이핑 스크립트입니다.

동적 페이지 로딩을 처리하며 공고의 '기술 스택 태그'와 '본문 텍스트'를 수집합니다.

정규표현식(re)을 사용하여 "R&D"와 같은 노이즈 데이터를 필터링합니다.

### visualizer.py

수집된 데이터를 정제하고 시각화하는 파이프라인입니다.

- 데이터 정규화: 'React.js' -> 'React', 'Amazon Web Services' -> 'AWS' 등 동일 기술의 표기를 통일합니다.

- 시각화: Matplotlib과 Seaborn을 활용하여 막대그래프(Bar Chart), 히트맵(Heatmap), 누적 막대그래프(Stacked Bar Chart)를 생성합니다.

### wanted_jobs.csv

수집 및 1차 전처리가 완료된 원본 데이터 파일입니다.

### graph_*.png

분석 결과로 도출된 최종 시각화 이미지 파일들입니다.

## 4. 요구 사항 (Requirements)

프로젝트 실행을 위해 다음 라이브러리가 필요합니다.

- python >= 3.10

- pandas

- selenium

- beautifulsoup4

- webdriver-manager

- matplotlib

- seaborn

### 설치 명령어:
```
pip install pandas selenium beautifulsoup4 webdriver-manager matplotlib seaborn
```

## 5. 시각화 결과 및 분석

### 분석 1. 직무별 핵심 요구 기술 (Top Skills)

- 서버 개발자: **Java(36.4%)** 와 **AWS(36.4%)** 가 공동 1위. Spring, MySQL이 뒤를 이으며 백엔드 생태계의 표준을 보여줍니다.

- 프론트엔드 개발자: **React(71.7%)** 가 압도적 1위. TypeScript(47.8%) 역시 높은 비중을 차지하여 사실상의 필수 기술임을 입증했습니다.

- 데이터 사이언티스트: **Python(65.5%)** 이 지배적이며, PyTorch(44.8%), TensorFlow(31.0%) 등 딥러닝 프레임워크 요구 비중이 매우 높습니다.

### 분석 2. 직무 간 기술 교차 비교 (Heatmap)

- **공유 기술 (Shared Skills)**: Git과 SQL은 직무와 무관하게 10~20%대 비중으로 꾸준히 요구되어, 개발자의 기초 소양임을 확인했습니다.

- **특화 기술 (Specialized Skills)**: 히트맵 상에서 직무별로 색상이 진한 영역이 명확히 분리되었습니다. 이는 '개발자'라는 큰 범주 안에서도 직무 간 기술 장벽이 존재함을 의미합니다.

### 분석 3. 기술 유형별 비중 분석 (Category Analysis)

- 서버 개발자: 인프라/클라우드 비중(23.1%)이 높아 DevOps(운영) 역량의 중요성이 확인되었습니다.

- 프론트엔드 개발자: 라이브러리/프레임워크 비중(44.3%)이 가장 높아 특정 도구(React 등) 숙련도가 중요합니다.

- 데이터 사이언티스트: 언어(Language)와 라이브러리 비중이 균형을 이루며, 도구보다는 언어 자체와 분석 능력을 중시합니다.

## 6. 결론 및 제언

본 분석을 통해 신입 채용 시장에서의 **'기술 파편화'** 와 **'직무 전문화'** 경향을 확인했습니다.

- 서버 개발자: 코드 구현을 넘어 AWS, Docker 등 배포/운영 능력(DevOps) 함양 필요

- 프론트엔드 개발자: React와 TypeScript가 표준으로 정착됨에 따라, 라이브러리 및 프레임워크 활용 능력의 고도화 필요

- 데이터 사이언티스트: 적은 공고 수(N=29)와 높은 기술 장벽(딥러닝 등)을 고려한 고도화된 역량 확보 전략 필요

최종적으로, 본 프로젝트의 분석 파이프라인을 확장하여 예비 개발자에게 개인화된 학습 로드맵을 제공하는 '기술 갭(Gap) 분석 솔루션' 개발을 제안합니다.
